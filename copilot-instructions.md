# Copilot Instructions for EECalculator

## Project Overview

Electrical Engineering Calculator - A comprehensive web-based tool for calculating electrical parameters and circuit analysis.

## Project Structure

- `index.html` - Main application with all calculator pages
- `css/style.css` - Responsive styling and design
- `js/calculations.js` - Mathematical calculations and functionality
- `README.md` - Project documentation

## Core Features

1. **Resistance Calculations** - Series and parallel equivalent resistance
2. **Capacitance Calculations** - Series and parallel capacitance
3. **Inductance Calculations** - Series and parallel inductance
4. **RC Circuit Analysis** - Transient voltage calculations
5. **RL Circuit Analysis** - Transient voltage calculations

## Key Characteristics

- No external dependencies (pure HTML/CSS/JavaScript)
- Responsive design for all devices
- Input validation and error handling
- Real-time calculations with up to 6 decimal places
- Portuguese language interface

## Maintenance Guidelines

- All calculations use standard electrical engineering formulas
- Input validation is mandatory for all numeric fields
- Error messages should be user-friendly and in Portuguese
- CSS maintains professional appearance with gradient backgrounds
- JavaScript functions are organized by circuit type

## Notes

- Project is located in c:\Sites\EECalculator\
- All formulas verified against standard electrical theory
- Unit conventions: Ω (ohms), F (farads), H (henries), V (volts), s (seconds)
